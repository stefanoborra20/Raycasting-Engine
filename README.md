# Raycasting Engine

A minimal raycasting engine made in C and [SDL2](https://www.libsdl.org).

## Build/Run/Debug
```console
make
./rc
```
or
```console
make debug
./rc
```
for debug mode.
## Features

- 2D map view
- 3D map view
- Resizable window


## Usage/Examples

- [v] Switch 2D/3D mode
- [p] Enable/disable squares outlines
- [q] Quit engine

### 2D mode
- [WASD] Movement
- [Mouse] View direction

### 3D mode
- [w] Up
- [s] Down
- [a/d] Turn view left/right
