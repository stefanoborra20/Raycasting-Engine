# Raycasting Engine

A minimal raycasting engine made in C and [SDL2](https://www.libsdl.org).

## Build/Run
```console
make
./rc
```
## Features

- 2D map view
- 3D map view
- Resizable window


## Usage/Examples

- [v] switch 2D/3D mode
- [p] enable/disable squares outlines
- [q] quit engine

### 2D mode
- [WASD] movement
- [Mouse] view direction
